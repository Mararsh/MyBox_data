# MyBox的数据
这里存放与MyBox有关的数据资源。

“MyBox内部格式”和“MyBox外部格式”可导入MyBox v6.3及以后的版本。   
MyBox内部格式：外键引用数据标识。好处是快速导入/导出，坏处是数据相互依赖。   
MyBox外部格式：数据都是显值。好处是自洽、可用于其它软件，坏处是导入/导出很慢。
